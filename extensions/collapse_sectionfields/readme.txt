Collapse Sectionfields
-------------------------------------------------------------------------------

Version: 1.0.1
Author: Carsten de Vries <carsten@vrieswerk.nl>
Build Date: 11 July 2009
Github Repository: http://github.com/carsten/globalparamloader/tree/master
Requirements: Symphony 2

Adds a link to collapse all fields in section edit pages, which makes ordering 
a large amount of sectionfields a lot easier.

Installation
-------------------------------------------------------------------------------

1. Upload the 'collapse_sectionfields' folder in this archive to your Symphony
   'extensions' folder.

2. Enable it by selecting the "Collapse Sectionfields", choose Enable from the
   with-selected menu, then click Apply.

3. Section edit pages now have a link to collapse/uncollapse sectionfields.
